Activiti Release Guide

Release day - 1 week

Code freeze. Ask on the dev forum who has outstanding work and synchronize on creating the release branch.

Release day

  1. Create the release branch: svn -m "creating release branch for 5.0.alpha3" copy https://svn.activiti.org/repo/activiti/trunk https://svn.activiti.org/repo/activiti/tags/activiti-5.0.alpha3
  2. Update the version in the release branch: s/5.0.alpha3-SNAPSHOT/5.0.alpha3/g
  3. Verification: Run the test suite: mvn -Pcheck clean install in the code base root
  4. Verification: Run the demo setup: ant test.demo.setup in directory qa
  5. Build the distribution: ant clean distro in directory distro
  6. Upload the distribution: TODO
  7. Publish the docs: ant publish.docs in directory distro
  8. Add the new release to the downloads page in the website
  9. Commit the docs and download page updates to the website
  10. Upload the maven artifacts to the repository (see dev guide)
  11. Blog and tweet about it
